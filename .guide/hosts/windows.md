# windows.md

## Use Cases

- Claude runs on the host
- Codex runs on the host
- VS Code runs on the host
- Chrome runs on the host for administrative and operational workflows
- OBS runs on the host for recording

## Constraints

- Keep the host available for interactive development work.
- Keep the host available for operator-driven browser workflows.
- Keep the host available for recording workflows.
- Keep Claude execution available on the host.
- Keep Codex execution available on the host.
- Keep editor workflows available on the host.
- Keep browser workflows available on the host.
- Keep recording workflows available on the host.
